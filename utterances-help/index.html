2.1 安装——为你的仓库装上 utterancesutterancesutterances

还有，不仅仅是 GrideaGrideaGridea 可以装 utterancesutterancesutterances ，任何用 Github PagesGithub\text{ }PagesGithub Pages 搭建的博客都可以安装 utterancesutterancesutterances 。

2.2 配置——获得你的专有 utterancesutterancesutterances 代码

安装完了，接下来进入这个网站进行配置。

英文不好？没事，我来告诉你。

  1. 打开网页。
  2. Ctrl+fCtrl+fCtrl+f 搜索repo:，应该只有一个。
  3. 再 repo: 下面的输入框输入你的博客用户的仓库，例如我的就是 GarbageBlog/Garbage.github.io
  4. 往下走一点，看到 Choose the mapping between blog posts and GitHub issues. 这行字，这里是要你选择 GithubIssuesGithub IssuesGithubIssues 的配置方式，除非你明确它是什么意思，否则就用第一个\color{red}\text{除非你明确它是什么意思，否则就用第一个}除非你明确它是什么意思，否则就用第一个 。
  5. 往下翻页，找到 label (optional): 这里，下面有个输入框，它的作用是将是评论的 GithubIssuesGithub IssuesGithubIssues 打上你在输入框输入的标签，标签必须在 Github IssuesGithub\text{ }IssuesGithub Issues 中存在！。(我是不会告诉你标签可以是中文的)
  6. 最后配置 Theme 项，就是选择主题颜色了，这个自己定。选完之后背景会变成你选的那个。
  7. 获取代码！配置完之后，点一下 Enable Utterances 中的 CopyCopyCopy 按钮，代码就复制成功了！
